# Package index

## Main functions

Essential functionality

- [`add_stats()`](https://willemsleegers.github.io/tidystats/reference/add_stats.md)
  : Add statistical output to a tidystats list

- [`write_stats()`](https://willemsleegers.github.io/tidystats/reference/write_stats.md)
  : Write a tidystats list to a file

- [`read_stats()`](https://willemsleegers.github.io/tidystats/reference/read_stats.md)
  :

  Read a .json file that was produced with
  [`write_stats()`](https://willemsleegers.github.io/tidystats/reference/write_stats.html)

- [`custom_stat()`](https://willemsleegers.github.io/tidystats/reference/custom_stat.md)
  : Create a custom statistic

- [`custom_stats()`](https://willemsleegers.github.io/tidystats/reference/custom_stats.md)
  : Create a collection of custom statistics

- [`tidy_stats_to_data_frame()`](https://willemsleegers.github.io/tidystats/reference/tidy_stats_to_data_frame.md)
  : Convert a tidystats list to a data frame

## Analysis functions

Functions for performing analyses such as calculating descriptives

- [`count_data()`](https://willemsleegers.github.io/tidystats/reference/count_data.md)
  : Count the number of observations
- [`describe_data()`](https://willemsleegers.github.io/tidystats/reference/describe_data.md)
  : Calculate common descriptive statistics
